Install¶

Prerequisites¶

  • A love for the command line

  • Python 3

  • Pip

Note

The Python modules Urwid, Requests, and Simperium3 will also be installed.

Installation¶

Pip¶

$ pip install sncli

Arch AUR¶

sncli is available in AUR as sncli-git.

Manually¶

$ git clone https://github.com/insanum/sncli.git
$ cd sncli/
$ python setup.py install

Using Pipenv¶

$ git clone https://github.com/insanum/sncli.git
$ cd sncli/
$ pipenv install
$ pipenv run sncli
