CLI Usage
#########

Common
======

Notes List
==========

Pager View
==========




.. option:: h
  
  View the ``help`` menu

.. option:: q

  Quit ``sncli``

.. option:: S

  Perform a bi-directional ync local notes to Simplenote server.

.. option:: j

  Scroll down one line.

.. option:: k

  Scroll up one line.

.. option:: space

  Page down.

.. option:: b

  Page up.

.. option:: ctrl+d

  Scroll half a page down.

.. option:: ctrl+u

  Scroll half a page up.

.. option:: G

  Jump to the bottom of the page.

.. option:: g

  Jump to the top of the page.

.. option:: s

  Toggle the Status bar.

.. option:: C

  Create a new note.

.. option:: e

  Edit highlighted note.

.. option:: enter

  Open highlighted note in sncli pager.

.. option:: meta+enter

  Open highlighted note in ???

.. option:: O

  View the selected note's JSON in ``cfg_editor``.

.. option:: |

  Pip note to another program.

.. option:: J

	View the next note.

.. option:: K

	View the previous note.

.. option:: l

	View the sync log.

.. option:: 2

	Tabstop 2

.. option:: 4

	Tabstop 4

.. option:: 8

 Tabstop 8

.. option:: <

	View an old version of the note (this cycles).

.. option:: >

	View a newer version of the note (this cycles).
